# 📚 Enhanced Book Management Application

This project is a feature-rich **Book Management System** developed using **ASP.NET Core MVC** and **Entity Framework**. It provides both admin and member functionalities for managing a digital library efficiently.

---

## 🚀 Features

### 👨‍💼 Admin Panel
- Add, edit, and delete book records.
- Manage members with validation.
- Assign or unassign books to users.
- Track book quantities and availability.

### 👤 Member Panel
- Search books by **title**, **author**, and **availability**.
- Borrow and return books with **real-time status updates**.

### 🔐 Authentication
- Secure registration and login with a user-friendly interface.
- Role-based access for **Admins** and **Members**.
